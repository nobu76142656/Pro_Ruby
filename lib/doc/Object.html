    convert_length(length, from: :m, to: :m) click to toggle source

    上記をキーワード引数を使うようにする

    # File convert_length.rb, line 22
def convert_length(length, from: :m, to: :m)
  units = { m: 1.0, ft: 3.28, in: 39.37 }
  #変換前の単位の長さ/変換前の単位の比率*変換後の単位の比
  (length / units[from] * units[to]).round(2)
end

    to_hex(r, g, b) click to toggle source

    inject(たたみ込みメソッド)を使ってさらに短くする。 ブロックの戻り値がinjectメソッド自身の戻り値となる。

    # File rgb.rb, line 26
def to_hex(r, g, b)
  [r, g, b].inject('#') do |hex, n|
    hex + n.to_s(16).rjust(2, '0')
  end
end
    to_ints(hex) click to toggle source

    mapメソッドを使ってリファクタリング mapメソッド：空の配列を用意して、他の配列をループ処理した結果を 空の配列に入れていく処理の大半はmapメソッドで置き換えられる。

    # File rgb.rb, line 66
def to_ints(hex)
  r = hex[1..2]
  g = hex[3..4]
  b = hex[5..6]
  [r, g, b].map do |s|
    s.hex
  end
end
